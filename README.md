Aim Trainer - Shooting Range

Aim Trainer adalah game latihan aim berbasis web menggunakan JavaScript dan Three.js yang dirancang untuk melatih akurasi, refleks, dan kecepatan pemain dalam menembak target. Project ini dibuat sebagai bagian dari pembelajaran Grafika Komputer dengan konsep FPS (First Person Shooter) aim training.

Features
Multiple Game Modes
Spidershot
Motionshot
Practice Mode
Real-time Score System
Accuracy Tracking
Countdown Timer
Pause Menu
Animated Background
FPS-style Camera Movement
Responsive HUD Interface
Difficulty Selection
Motion Speed & Distance Settings
Modern UI Design
Technologies Used
HTML5
CSS3
JavaScript
Three.js
Vite
Cara Menjalankan Project
1. Clone Repository
git clone https://github.com/USERNAME/REPOSITORY.git
2. Masuk ke Folder Project
cd REPOSITORY
3. Install Dependency

Pastikan sudah menginstall Node.js terlebih dahulu.

Install dependency menggunakan:

npm install
4. Jalankan Project
npm run dev
5. Buka di Browser

Setelah dijalankan, terminal akan menampilkan link seperti:

http://localhost:5173

Buka link tersebut di browser.

Controls
Key / Action	Function
Mouse Movement	Aim
Left Click	Shoot
ESC	Pause Menu
Game Modes
Spidershot

Mode latihan aim klasik dengan target statis yang muncul secara acak.

Motionshot

Mode target bergerak dengan pengaturan speed dan distance untuk meningkatkan tracking aim.

Practice

Mode bebas tanpa batas waktu untuk latihan santai.
